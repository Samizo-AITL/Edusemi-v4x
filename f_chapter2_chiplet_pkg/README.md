# 📦 特別編 第2章：チップレットと先端パッケージ技術

## 🧭 概要

本章では、モノリシックSoCの微細化限界やコスト上昇を背景に注目される「チップレット」技術と、それを支える先端パッケージング（2.5D/3D実装）について解説します。  
TSVやインターポーザを活用した実装手法、実際の商用事例、設計課題、そして技術選定のポイントを体系的にまとめます。
