## 📄 論文概要｜Paper Overview
EDA設計フローにPID+FSM+LLM制御ループを統合し、  
RC遅延・熱結合・EMI変動を実時間で補償する枠組みを提案。  
sub-2nmノードにおけるガードバンド削減と信頼性向上を実証する。  

Cross-layer runtime adaptation enables guardband reduction and improved reliability across sub-2nm nodes.
