# 📘 0.18μm FeRAM Process Flow（強誘電体メモリプロセス）解説版

[![Hybrid License](https://img.shields.io/badge/license-Hybrid-blueviolet)](https://samizo-aitl.github.io/Edusemi-v4x/#-ライセンス--license)

## 🧭 概要 / Overview

本教材では、**0.18μm CMOSロジックプロセス**を基盤とした仮想構成により、**FeRAM（強誘電体メモリ）**の代表的な製造フローを解説します。  
キャパシタ構造には **Pt/PZT/Ti** を採用し、**Coサリサイド（CoSi₂）**、**多層Al配線（Metal-1〜3）＋Wプラグ接続構造**など、実用プロセス技術と整合性のある構成です。  

また、本構成では、**メモリセル部（3.3V）とロジック部（1.8V）の二電源構成**を想定しています。  
これは、**FeRAMセルの分極反転に十分な高電圧が必要である一方、ロジック回路は低電圧で動作させて消費電力と信頼性を確保するため**です。  

---

This document presents an educational process flow for FeRAM, constructed on a **0.18μm CMOS logic platform**.  
The capacitor stack is composed of **Pt/PZT/Ti**, and the flow includes **Co salicide (CoSi₂)**, **triple-layer Al interconnects with W-plug via structure**, and other practical fabrication steps.  

The design assumes a **dual-voltage configuration** with **3.3V for memory cells** and **1.8V for logic circuits**.  
This is because **FeRAM cells require a sufficiently high voltage for polarization switching, while logic circuits operate at lower voltage for reduced power consumption and improved reliability**.

## 📋 プロセスフロー要約 / *Process Flow Summary*  

| 工程カテゴリ / *Process Category* | 主な処理 / *Main Process* | 目的 / *Purpose* |
|-----------------------------------|---------------------------|------------------|
| **素子分離 / Isolation** | STI形成、CMP平坦化、犠牲酸化膜 <br>*STI formation, CMP planarization, sacrificial oxidation* | 素子間リーク防止・注入前表面改質 <br>*Isolation and surface preparation before implantation* |
| **ウェル・チャネル / Well & Channel** | N/Pウェル形成、チャネル注入 <br>*N/P well formation, channel doping* | デバイス基盤形成、Vth調整 <br>*Well structure and threshold control* |
| **ゲート・ソースドレイン / Gate & S/D** | G1/G2酸化、Poly-Siゲート形成、LDD、深拡散、CoSi₂サリサイド <br>*G1/G2 oxidation, poly-Si gate, LDD, S/D, CoSi₂ salicide* | CMOSデバイス完成 <br>*Complete CMOS transistor formation* |
| **コンタクト層 / Contact & 2nd ILD** | ILD堆積、コンタクト開口、TiNバリア、Wプラグ形成 <br>*ILD deposition, contact via, TiN barrier, W plug* | デバイス〜配線の接続 <br>*Connect device to interconnect layers* |
| **キャパシタ / FeRAM Capacitor** | Pt/PZT/Tiキャパシタ積層、AlOx保護（二重膜） <br>*Pt/PZT/Ti capacitor stack, dual AlOx protection* | FeRAMセル形成 <br>*FeRAM cell formation* |
| **BEOL配線 / BEOL Interconnects** | Al配線（M1〜M3）、Wプラグ、CMP平坦化 <br>*Al interconnects (M1–M3), W plugs, CMP* | 多層配線接続 <br>*Multilayer interconnect wiring* |
| **Pad・Passivation** | 厚膜Alパッド、SiN/SiO₂パッシベーション <br>*Thick Al pads, SiN/SiO₂ passivation* | 外部I/O保護、長期信頼性 <br>*External I/O protection, long-term reliability* <br>※ 通常の水素シンター工程は **PZT還元劣化防止のため削除** <br>*Hydrogen sintering omitted to avoid PZT degradation* |

## 🧠 FeRAM特有工程の詳細解説 / *Detailed Explanation of Key FeRAM Steps*

---

### 🔽 キャパシタ形成（Pt/PZT/Ti） / *Capacitor Formation (Pt/PZT/Ti)*  

| 工程 / Step | 内容 / Description |
|-------------|--------------------|
| **TI1-SP（下部）** | TiスパッタによるPt密着層（約300Å）。<br>*Ti sputtering adhesion layer for Pt (~300Å).* |
| **Pt-SP** | Pt下部電極（1000–1500Å）、高耐久性・導電性。<br>*Pt bottom electrode (1000–1500Å), durable and conductive.* |
| **PZT-COT / PZT-ANL** | PZTスピンコート＋アニール、ペロブスカイト結晶化。<br>*PZT spin coating + annealing for perovskite crystallization.* |
| **TI2-SP（上部）** | 上部電極、Al配線とのバッファ機能。<br>*Top electrode, also buffer to Al wiring.* |
| **結晶配向確認** | 上部電極形成後、XRDでPZTの(100)/(111)面配向を確認。<br>*After top electrode, XRD verifies PZT (100)/(111) orientation.* |
| **CAP-PH / CAP-ET** | Pt/PZT/Tiパターニング。PtはCMP不可、IBE等の物理エッチ必要。<br>*Pt/PZT/Ti patterning; CMP not applicable, requires IBE/physical etching.* |

---

### 🔽 保護膜形成（AlOx） / *Protective Layer Formation (AlOx)*  

| 工程 / Step | 内容 / Description |
|-------------|--------------------|
| **ALOX-SP / ALOX-DP** | スパッタ＋ALDによるAlOx保護膜（二重膜）、ピンホール制御が鍵。<br>*AlOx protection by sputter + ALD (dual layer); pinhole suppression is key.* |
| **ALOX-PH / ALOX-ET** | 開口形成、後続配線層との接続用。<br>*Via opening for subsequent interconnect.* |

---

### 🔽 接続構造（Metal-0〜Via-0） / *Interconnect Structure (Metal-0 to Via-0)*  

| 工程 / Step | 内容 / Description |
|-------------|--------------------|
| **HLX-DP** | PE-TEOSによるILD-0堆積。<br>*ILD-0 deposition using PE-TEOS.* |
| **HLX-PH / HLX-ET** | Via-0開口形成。<br>*Lithography + etching for Via-0.* |
| **TINX-SP** | Ti/TiNバリア層形成（約300Å）。W拡散防止。<br>*Ti/TiN barrier (~300Å), prevents W diffusion.* |
| **HWX-DP / HWX-CMP** | Wプラグ充填とCMP平坦化。キャパシタ→M1接続を確立。<br>*W plug fill + CMP; connects capacitor to Metal-1.* |

---

### ⚠️ 水素還元によるPZT特性劣化対策 / *Mitigation of PZT Degradation by H₂*  

| 項目 / Item | 内容 / Description |
|-------------|--------------------|
| **懸念事項** | 水素還元によりPZTのヒステリシス特性が劣化（インプリント）。<br>*Hydrogen reduction degrades PZT hysteresis (imprint).* |
| **対策** | AlOx二重保護膜で隔離。最終の水素シンター工程を削除。<br>*Dual AlOx layers isolate PZT; final hydrogen sintering excluded.* |
| **改善** | HfZrO₂採用で水素耐性向上。420℃ H₂シンター対応可能。<br>*HfZrO₂ provides hydrogen resistance, enables 420℃ H₂ sintering.* |

👉 詳細は [技術ノート HfO₂系の水素対応性](../../d_chapter1_memory_technologies/doc_FeRAM/hfo2_h2_compat.md) を参照。  
👉 For details, see [Tech Note: HfO₂ Hydrogen Compatibility](../../d_chapter1_memory_technologies/doc_FeRAM/hfo2_h2_compat.md).

### 📘 特性評価 / *Property Evaluation*  

| 評価手法 / Method | 内容 / Description |
|-------------------|--------------------|
| **ヒステリシス測定 (PUND法)** | Pm, Pr, Vcを測定。FeRAM保持特性を評価。<br>*Measures Pm, Pr, Vc using PUND method; evaluates retention.* |
| **バタフライカーブ** | PZTアクチュエータの変位特性。<br>*Voltage–displacement curve of PZT actuators.* |
| **DBLI法** | 二重ビームレーザー干渉法による変位評価。<br>*Displacement measured by Double Beam Laser Interferometry (DBLI).* |

👉 詳細は [FeRAMピエゾ評価法](./feram_piezo_evaluation_principles.md) を参照。  
👉 See [FeRAM Piezo Evaluation Principles](./feram_piezo_evaluation_principles.md) for details.

### 📝 補足 / *Notes*  

- **PZT膜のZr/Ti比と焼成条件**は、スイッチング特性・保持特性に大きく影響。  
  *Zr/Ti ratio and annealing conditions of the PZT film greatly affect switching and retention properties.*  

- **AlOx膜の品質（密度・膜厚均一性）**は長期信頼性に直結。  
  *Quality of AlOx film (density and thickness uniformity) directly impacts long-term reliability.*
