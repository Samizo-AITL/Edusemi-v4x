# 🎛️ 応用編 第5章：アナログ／ミックスドシグナル

---

## 📘 概要

アナログ／ミックスドシグナル（AMS）設計は、デジタルとアナログが混在するSoCやセンサ応用において不可欠な技術領域です。  
本章では、オペアンプやコンパレータなどの基本回路から、レイアウト設計、ノイズ対策、ADC/DACの混載課題に至るまで、AMS設計の全体像を構造的に学びます。
